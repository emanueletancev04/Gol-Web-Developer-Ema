ELEMENTI DI TESTO: Headings (elementi block)

h1 - Titolo principale - 32px

Deve essere presente SOLTANTO UNA VOLTA in una pagina

h2 - Sottotitolo se IMMEDIATAMENTE successivo a un h1, altrimenti titolo di un contenuto successivo - 24px

h3 - Sottotitolo di un h2, poco usato per il corpo del testo - 18/19px (18.72)

h4 - Generalmente corpo del testo per piccoli blocchi di contenuto - 16px (dimensione predefinita del testo)

h5 - Generalmente testo secondario - 13/14px (13.28)
h6 - Testo in nota, poco usato per le dimensioni - 10/11px (10.72)

p (paragraph) - Testo normale o corpo del testo - 16px

ELEMENTI DI TESTO: Liste (elementi block)

Liste non ordinate con ul, liste ordinate con ol, elementi di lista con li, elementi di lista con il contenuto vero e proprio, formattato con la dimensione predefinita del testo (16px)

Lista non ordinata: blocco ul

  • Primo elemento
  • Secondo elemento
  • Terzo elemento

Lista ordinata: blocco ol

  1. Primo elemento
  2. Secondo elemento
  3. Terzo elemento

Lista complessa

  • Primo elemento
  • Secondo elemento
    • Primo sottoelemento
    • Secondo sottoelemento
  • Terzo elemento
  1. Primo elemento
  2. Secondo elemento
    1. Primo sottoelemento
    2. Secondo sottoelemento
  3. Terzo elemento
